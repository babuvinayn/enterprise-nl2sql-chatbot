# Enterprise NL2SQL Chatbot

An AI-powered Natural Language to SQL backend system built using FastAPI and Gemini.

## Features
- Semantic table prediction
- FAISS-based similarity search
- Gemini LLM integration
- Dynamic SQL generation
- Modular backend architecture

## Tech Stack
- Python
- FastAPI
- FAISS
- Google Gemini
- Sentence Transformers
